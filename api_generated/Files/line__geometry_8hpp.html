function get_segment_between_percentages

Line2D get_segment_between_percentages(
    Line2D const & line,
    float start_percentage,
    float end_percentage
)

Extract a segment of a line between two percentage points along its cumulative distance.

Parameters:

  • line The input line (vector of Point2D)
  • start_percentage Start percentage (0.0 to 1.0, where 0.0 is beginning, 1.0 is end)
  • end_percentage End percentage (0.0 to 1.0, where 0.0 is beginning, 1.0 is end)

Return: Line2D containing the extracted segment

Note:

  • If start_percentage >= end_percentage, returns empty line
  • If line is empty or has only one point, returns empty line
  • Percentages are clamped to [0.0, 1.0] range

This function calculates the cumulative distance along the line and extracts a continuous segment between the specified start and end percentages.

function calculate_perpendicular_direction

Point2D< float > calculate_perpendicular_direction(
    Line2D const & line,
    size_t vertex_index
)

Calculate the perpendicular direction at a specific vertex of a line.

Parameters:

  • line The line to calculate perpendicular direction for
  • vertex_index The index of the vertex (0 to line.size()-1)

Return: Point2D representing the normalized perpendicular direction vector

Note:

  • Returns (0,0) if line has fewer than 2 points or vertex_index is invalid
  • For edge vertices, uses the single adjacent segment
  • For internal vertices, averages perpendiculars from both adjacent segments

For internal vertices (1 to n-1), calculates the average of perpendicular directions from adjacent segments. For edge vertices (0 and n-1), calculates the perpendicular of the single adjacent segment.

function is_distal_end_closer

bool is_distal_end_closer(
    Line2D const & line,
    Point2D< float > const & reference_point
)

Determine if the distal (last) point of a line is closer to a reference point.

Parameters:

  • line The line to evaluate
  • reference_point The point to measure distance to

Return:

  • true if the last point is closer to reference_point than the first point
  • false if the first point is closer or equal, or if line has fewer than 2 points

Compares the squared distance from both endpoints to the reference point. Returns true if the last point is strictly closer than the first point.
