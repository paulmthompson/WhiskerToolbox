function rememberPath

void rememberPath(
    QString const & dialog_id,
    QString const & path
)

Remember a path for a specific dialog.

Parameters:

  • dialog_id Unique identifier for the dialog
  • path File or directory path to remember

If the path is a file, the parent directory is stored instead. This auto-saves after a debounce period.
