function setDockStateCaptureCallback

void setDockStateCaptureCallback(
    std::function< std::string()> callback
)

Set a callback to capture ADS dock state for layout persistence.

The callback should return a base64-encoded string of the ADS CDockManager::saveState() QByteArray. This avoids coupling WorkspaceManager to the ADS library.

function saveWorkspace

bool saveWorkspace(
    QString const & file_path
)

Save the current session to a workspace file.

Parameters:

  • file_path Absolute path to the workspace file

Return: true on success

Captures editor states, zone layout, and data load entries. Updates the current workspace path and clears the dirty flag.

function readWorkspace

std::optional< WorkspaceData > readWorkspace(
    QString const & file_path
) const

Load a workspace file and return its data.

Parameters:

  • file_path Path to the workspace file

Return: WorkspaceData on success, std::nullopt on failure

Parses the workspace file and returns the data. The caller (MainWindow) is responsible for orchestrating the actual restore (data reload, state restore, widget recreation, layout application).

function markRestored

void markRestored(

Mark the workspace as successfully restored.

Sets the current path and clears dirty state after a successful restore orchestrated by the caller.
