Detailed Description

struct ScatterColorConfigData;

Serializable configuration for external feature-based point coloring.

When color_mode is “fixed”, points use the default GlyphStyleData color. When “by_feature”, points are colored by resolving color_source against the DataManager and mapping the resulting float values according to mapping_mode.
