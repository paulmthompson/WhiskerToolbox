Detailed Description

class LineTimestampComputer;

Computer that extracts timestamps from line data.

Source type: LineData Selector type: Timestamp Output type: int64_t

Given a line source and a Timestamp-based ExecutionPlan, extracts the timestamp value for each line. Multiple lines at the same timestamp are expanded into separate rows, each with the same timestamp value.
