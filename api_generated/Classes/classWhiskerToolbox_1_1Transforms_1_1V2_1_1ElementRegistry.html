Detailed Description

class WhiskerToolbox::Transforms::V2::ElementRegistry;

Registry for element-level transforms.

Maintains a compile-time typed registry of transforms that operate on individual elements (Mask2D, Line2D, float, etc.).

Features:

  • Type-safe registration and lookup
  • Query by input/output types
  • Automatic container lifting
  • Metadata for UI generation

Register a container-level transform.

Parameters:

  • name Unique transform name
  • func Transform function: (Container const&, Params const&, Context const&) -> shared_ptr
  • metadata Optional metadata

Template Parameters:

  • InContainer Input container type (e.g., AnalogTimeSeries)
  • OutContainer Output container type (e.g., DigitalEventSeries)
  • Params Parameter type

These are transforms that need full container access (temporal dependencies, etc.) Transform function should be pure: take concrete types, return concrete types. No virtual inheritance, no variants - pipeline handles type erasure.
