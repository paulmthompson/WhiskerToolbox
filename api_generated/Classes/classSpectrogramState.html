Detailed Description

class SpectrogramState;

State class for SpectrogramWidget.

SpectrogramState is the Qt wrapper around SpectrogramStateData that provides typed accessors and Qt signals for all state properties.

Public Signals Documentation

Emitted when any view state property changes (zoom, pan, bounds)

function setXZoom

void setXZoom(
    double zoom
)

Set X-axis zoom factor (view transform only)

Parameters:

  • zoom Zoom factor (1.0 = full window visible, 2.0 = half visible)

Adjusts magnification without changing the underlying data window. Only emits viewStateChanged(), not stateChanged() - no scene rebuild.

function setYZoom

void setYZoom(
    double zoom
)

Set Y-axis zoom factor (view transform only)

Parameters:

  • zoom Zoom factor (1.0 = all frequencies fit, 2.0 = half the frequencies fit)

Adjusts frequency density without changing the underlying data. Only emits viewStateChanged(), not stateChanged() - no scene rebuild.

function setPan

void setPan(
    double x_pan,
    double y_pan
)

Set pan offset (view transform only)

Parameters:

  • x_pan X-axis pan offset in world units (ms)
  • y_pan Y-axis pan offset in normalized units

Shifts the view without changing the underlying data window. Only emits viewStateChanged(), not stateChanged() - no scene rebuild.

function setXBounds

void setXBounds(
    double x_min,
    double x_max
)

Set X-axis data bounds (triggers scene rebuild)

Parameters:

  • x_min Minimum time before alignment (typically negative)
  • x_max Maximum time after alignment (typically positive)

Changes the window of data gathered from the DataManager. Emits both viewStateChanged() AND stateChanged() - triggers scene rebuild.
