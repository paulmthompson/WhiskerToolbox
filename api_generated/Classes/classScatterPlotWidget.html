Public Slots Documentation

slot handleExportSVG

void handleExportSVG()

Open a file dialog and write the current scene to an SVG file.

Uses [AppFileDialog](/api_generated/Namespaces/namespaceAppFileDialog.md) for session-aware path memory (dialog id "export_scatter_plot_svg"). Shows an informational message box on success and an error message box on failure. Does nothing if the scene is empty or the user cancels the dialog.
