Detailed Description

class MaskInspector;

Inspector widget for MaskData.

Provides mask data inspection and management functionality including image size management and export capabilities.

function setActiveKey

virtual void setActiveKey(
    std::string const & key
) override

Set the active data key to inspect.

Parameters:

  • key The data key in DataManager

Reimplements: IDataInspector::setActiveKey

This method should:

  1. Remove callbacks from any previously active data
  2. Store the new key
  3. Set up callbacks on the new data
  4. Update the view to show the new data

function removeCallbacks

virtual void removeCallbacks() override

Remove all callbacks from the currently active data.

Reimplements: IDataInspector::removeCallbacks

Called when the inspector is being destroyed or when switching to a different data key. Must clean up any registered observers.
