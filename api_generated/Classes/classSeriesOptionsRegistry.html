Detailed Description

class SeriesOptionsRegistry;

Generic registry for all series display option types.

Provides a unified API for managing AnalogSeriesOptionsData, DigitalEventSeriesOptionsData, and DigitalIntervalSeriesOptionsData.

Get mutable display options for a key.

Parameters:

  • key The data key

Template Parameters:

Return: Pointer to options, or nullptr if not found

Use sparingly - prefer set() to ensure signals are emitted. Call notifyChanged() after modifying the returned options.
