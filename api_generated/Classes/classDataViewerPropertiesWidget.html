Detailed Description

class DataViewerPropertiesWidget;

Properties panel for Data Viewer Widget.

Displays feature tree, per-feature controls, and visualization settings. Shares state with DataViewer_Widget (view) via DataViewerState.

The view widget should connect to this signal to trigger auto-arrangement of all visible series.

signal seriesRelativePlacementRequested

void seriesRelativePlacementRequested(
    QString source_key,
    QString target_key,
    bool above
)

Emitted when the user requests relative placement of a series via the context menu.

Parameters:

  • source_key Series key to reposition
  • target_key Series key to place relative to
  • above true = place source above target, false = place below

The view widget should connect to this signal and translate it into deterministic lane_order overrides via DataViewerState.

signal saveLaneLayoutRequested

void saveLaneLayoutRequested()

Emitted when the user clicks “Save Lane Layout…”.

The view widget should connect to this signal to serialize the current lane placement and displayed series to a JSON file.

signal loadLaneLayoutRequested

void loadLaneLayoutRequested()

Emitted when the user clicks “Load Lane Layout…”.

The view widget should connect to this signal to restore lane placement and displayed series from a previously saved JSON file.

signal loadSpikeToAnalogPairingRequested

void loadSpikeToAnalogPairingRequested()

Emitted when the user clicks “Load Spike-to-Analog Pairing…”.

The view widget should connect to this signal to open the configuration dialog and apply the resulting lane placement overrides.
