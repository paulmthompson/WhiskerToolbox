Detailed Description

class dl::widget::SequenceSlotWidget;

Widget for configuring a single sequence (memory) input slot.

Note: Not thread-safe — must be used from the GUI thread only.

Manages a list of sequence entries, each driven by SequenceEntryVariant. Each entry can be Static (data from DataManager) or Recurrent (feedback from model output). Add/Remove buttons control the entry count. Capture button appears for Static+Absolute entries.
