Detailed Description

class PSTHState;

State class for PSTHWidget.

PSTHState is the Qt wrapper around PSTHStateData that provides typed accessors and Qt signals for all state properties.

PSTHState uses composition with PlotAlignmentState to provide alignment functionality via the shared PlotAlignmentWidget component.
