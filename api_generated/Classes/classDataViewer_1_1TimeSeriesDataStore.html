Detailed Description

class DataViewer::TimeSeriesDataStore;

Centralized storage for time series data in the DataViewer widget.

Manages storage, addition, removal, and configuration of analog, digital event, and digital interval time series. Provides signals for change notification and integrates with the CorePlotting layout system.

void applyGroupStdDev(
    std::vector< std::string > const & keys,
    float group_std_dev
)

Apply a unified standard deviation to a group of analog series.

Parameters:

  • keys Series keys in the group
  • group_std_dev Standard deviation to use for all series

Sets cached_std_dev and intrinsic_scale for all specified series to use the provided group_std_dev instead of their individual values. The individual_std_dev field is preserved for later reversion.
