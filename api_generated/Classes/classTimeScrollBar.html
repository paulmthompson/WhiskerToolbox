signal timeChanged

void timeChanged(
    TimePosition position
)

Emitted when user scrubs the timeline (includes TimeFrame pointer)

This is the preferred signal for time changes. It includes the TimePosition which contains both the index and the TimeFrame pointer for clock identity.

    EditorRegistry * registry
)

Set the EditorRegistry for time synchronization.

Parameters:

  • registry The EditorRegistry instance (can be nullptr)

TimeScrollBar will call EditorRegistry::setCurrentTime() when the user scrubs the timeline, ensuring all widgets stay synchronized.

function setKeymapManager

void setKeymapManager(
    KeymapSystem::KeymapManager * manager
)

Set the KeymapManager and register the adapter for timeline actions.

Parameters:

  • manager The KeymapManager to register with (must outlive TimeScrollBar)

Creates a KeyActionAdapter that handles play_pause, next_frame, prev_frame, jump_forward, and jump_backward actions dispatched by the KeymapSystem.

function setTimeFrame

void setTimeFrame(
    std::shared_ptr< TimeFrame > const & tf,
    TimeKey display_key =TimeKey("time")
)

Set which TimeFrame this scrollbar controls.

Parameters:

  • tf The TimeFrame to control (can be nullptr)
  • display_key The TimeKey for UI display (defaults to “time”)

Updates the scrollbar to control the specified TimeFrame. The display_key is used for UI labels (e.g., “Camera Time”, “Ephys Clock”).

function setFrameFilter

void setFrameFilter(
    std::unique_ptr< FrameFilter > filter
)

Install a frame filter for skip-frame-during-playback behaviour.

Parameters:

  • filter Owning pointer to the new filter, or nullptr to clear.

Replaces any previously installed filter. Pass nullptr to disable filtering. The filter’s TimeFrame is automatically updated when the scrollbar’s own TimeFrame changes.
