Detailed Description

class ScatterPlotState;

State class for ScatterPlotWidget.

Single source of truth: view_state (zoom/pan) plus horizontal and vertical axis states (full range). OpenGL widget and axis widgets read from state.

bool fromJson(std::string const& json) override { auto result = rfl::json::read(json); if (result) { _data = *result; emit stateChanged(); return true; } return false; } ```
