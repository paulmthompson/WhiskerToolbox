Detailed Description

class Media_Window;

The Media_Window class is responsible for plotting images, movies, and shapes on top of them. Shapes may take the form of lines, points, or arbitrary 2d masks. Advancing a frame will result in the video window loading new data.

This is a QGraphicsScene, which internally renders lines, paths, points, and shapes which are added to the scene

Shapes can be added to the specific frame being visualized after it has been rendered and saved or just temporarily (scrolled back it will not be there), or data assets can be loaded which are saved for the duration (loading keypoints to be plotted with each corresponding frame).
