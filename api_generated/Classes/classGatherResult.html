Detailed Description

template <typename T >
class GatherResult;

Container for collected views created by gather operations.

Template Parameters:

  • T The data type (e.g., DigitalEventSeries, AnalogTimeSeries)

GatherResult holds a collection of view-backed (or copied) data objects created by aligning a source to a set of intervals. It provides a range interface for iteration and convenience methods for analysis.

inline int64_t alignmentTimeAt(
    size_type i
) const

Get the alignment time for a specific trial (O(1) access)

Parameters:

  • i Index of the trial

Exceptions:

  • std::out_of_range if i >= size()

Return: Alignment time for trial i

Returns the time point used for alignment (t=0 reference) for the specified trial. This is the value that should be subtracted from event times to get trial-relative times.

For GatherResults created with:

  • IntervalWithAlignmentAdapter: Returns start, end, or center based on alignment setting
  • EventExpanderAdapter: Returns the event time (center of the expanded window)
  • Basic gather(): Returns interval.start as fallback

function buildTrialStore

inline WhiskerToolbox::Transforms::V2::PipelineValueStore buildTrialStore(
    size_type trial_idx
) const

Build value store for a specific trial (V2 pattern)

Creates a PipelineValueStore populated with standard trial values that can be bound to transform parameters. This enables generic parameter binding without specialized context structs.

Store Keys

  • “alignment_time”: int64_t - Trial start time (used as t=0 reference)
  • “trial_index”: int64_t - Original trial index (0-based)
  • “trial_duration”: int64_t - Duration (end - start)
  • “end_time”: int64_t - Trial end time

Usage with Pipeline Bindings

// Pipeline with bindings
// {"transform": "NormalizeTimeV2", "bindings": {"alignment_time": "alignment_time"}}
auto factory = bindValueProjectionV2<EventWithId, float>(pipeline);

for (size_t i = 0; i < result.size(); ++i) {
    auto store = result.buildTrialStore(i);
    auto projection = factory(store);
    // Use projection on trial events...
}

trial_idxIndex of the trial (0-based, respects reordering)

PipelineValueStore populated with trial values

std::out_of_rangeif trial_idx >= size()

PipelineValueStore for store documentation

projectV2() for applying store-based projections to all trials
