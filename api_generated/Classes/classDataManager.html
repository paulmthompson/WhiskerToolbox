function setTime

bool setTime(
    TimeKey const & key,
    std::shared_ptr< TimeFrame > timeframe,
    bool overwrite =false
)

Register a new temporal coordinate system with a unique key.

Parameters:

  • key The unique identifier for this TimeFrame
  • timeframe The TimeFrame object to register

Return: bool True if the TimeFrame was successfully registered, false otherwise

Note: If the key already exists or timeframe is nullptr, a warning message will be printed to std::cerr and the function will return false

This function stores a TimeFrame object in the DataManager under the provided key. The TimeFrame specifies the temporal coordinate system that can be assigned to data objects.

function reset

void reset()

Clear all data and reset DataManager to initial state.

Note: The default “time” TimeFrame and “media” data key are preserved but reset to empty state

This function removes all loaded data objects, TimeFrame objects (except the default “time” frame), and clears all mappings between data keys and time frame keys. This is useful for batch processing where you want to load different datasets with the same configuration.

The function will:

  • Remove all data objects from _data
  • Remove all TimeFrame objects except the default “time” frame
  • Clear all data-to-timeframe mappings
  • Remove all TimeFrameV2 objects and mappings
  • Reset media data to default empty state
  • Notify observers of the state change

function ensureTimeEntityId

EntityId ensureTimeEntityId(
    TimeKey const & time_key,
    TimeFrameIndex index
)

Register (or retrieve) an EntityId for a specific time point on a clock.

Parameters:

  • time_key The TimeFrame key (clock identity)
  • index The TimeFrameIndex within that clock

Return: The EntityId for this time point

Uses the TimeKey string as the entity’s data_key and EntityKind::TimeEntity. The local_index is always 0 since a TimeFrameIndex uniquely identifies a time point within a clock.

function getTimeIndicesInGroup

std::vector< TimeFrameIndex > getTimeIndicesInGroup(
    GroupId group_id,
    TimeKey const & time_key
) const

Get all TimeFrameIndex values in a group for a specific clock.

Parameters:

  • group_id The group to query
  • time_key The clock to filter by

Return: Vector of TimeFrameIndex values (unordered), empty if group doesn’t exist

Iterates the group’s entities, filters to TimeEntity entries whose data_key matches the given TimeKey, and returns their TimeFrameIndex values. Non-TimeEntity members and TimeEntities from other clocks are skipped.

function getTimeIndicesInGroupConverted

std::vector< TimeFrameIndex > getTimeIndicesInGroupConverted(
    GroupId group_id,
    TimeKey const & target_key
) const

Get all TimeFrameIndex values in a group, converted to a target clock.

Parameters:

  • group_id The group to query
  • target_key The target clock to convert indices into

Return: Vector of converted TimeFrameIndex values (unordered), empty if group doesn’t exist or target TimeFrame is not registered

For each TimeEntity in the group (regardless of its source clock), converts its index to the target clock using convert_time_index(). TimeEntities already on the target clock are included without conversion. Non-TimeEntity members are skipped. Requires that both the source and target TimeFrames are registered.
