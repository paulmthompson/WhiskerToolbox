Types Documentation

enum ColormapPreset

Enumerator Value Description
Inferno Dark-to-bright, perceptually uniform (default for heatmaps)
Viridis Blue-green-yellow, perceptually uniform (default for spectrograms)
Magma Dark-to-pink, perceptually uniform.
Plasma Blue-to-yellow, perceptually uniform.
Coolwarm Diverging blue-white-red (for z-scores)
Grayscale Simple black-to-white.
Hot Black-red-yellow-white (high contrast)

Available colormap presets.

Perceptually uniform sequential colormaps (Inferno, Viridis, Magma, Plasma) are from the matplotlib collection and are designed for accurate perception of magnitude differences. Coolwarm is a diverging colormap centered at white, suitable for signed data (e.g., z-scores).

function getColormap

ColormapFunction getColormap(
    ColormapPreset preset
)

Get a ColormapFunction for the given preset.

Parameters:

  • preset Colormap preset to use

Return: Callable that maps [0,1] → glm::vec4 (RGBA, alpha=1)

The returned function maps t ∈ [0,1] → RGBA using LUT interpolation. Values outside [0,1] are clamped.

function mapValue

glm::vec4 mapValue(
    ColormapFunction const & cmap,
    float value,
    float vmin,
    float vmax
)

Map a scalar value through a colormap with explicit range.

Parameters:

  • cmap Colormap function
  • value Scalar value to map
  • vmin Minimum of the data range
  • vmax Maximum of the data range

Return: RGBA color

Normalises value to [0,1] using (value - vmin) / (vmax - vmin), clamps the result, and evaluates the colormap.

Map a row-major matrix of scalar values to RGBA colors.

Parameters:

  • cmap Colormap function
  • values Flat row-major matrix of values (size = num_rows × num_cols)
  • num_rows Number of rows
  • num_cols Number of columns
  • vmin Minimum of the data range
  • vmax Maximum of the data range

Return: Vector of RGBA colors, size = num_rows × num_cols

Convenience wrapper around mapValues() — the matrix is treated as a flat array. The row/column parameters are provided for documentation clarity and future validation.
