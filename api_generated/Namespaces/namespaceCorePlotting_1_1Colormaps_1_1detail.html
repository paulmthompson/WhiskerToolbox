function generateHotLUT

inline ColormapLUT generateHotLUT()

Generate a Hot LUT from black (0) → red → yellow → white.

Follows the classic “hot” colormap: R ramps 0→1 over [0, 0.375], G ramps 0→1 over [0.375, 0.75], B ramps 0→1 over [0.75, 1.0].

function generateCoolwarmLUT

inline ColormapLUT generateCoolwarmLUT()

Generate a Coolwarm diverging LUT from blue (0) → white (0.5) → red (1)

Uses a simple smooth interpolation: blue at t=0, white at t=0.5, red at t=1. This is a simplified version of the Moreland diverging colormap.
