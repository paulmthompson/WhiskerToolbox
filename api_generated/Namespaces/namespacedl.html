enum RasterMode

Enumerator Value Description
Binary 1.0 at occupied pixels, 0.0 elsewhere
Heatmap Gaussian blob (requires sigma parameter)
Distance Distance transform from geometry.
Raw Direct pixel copy (images)

How a geometry primitive is rasterized onto a tensor channel.

enum BackendType

Enumerator Value Description
TorchScript .pt files via torch::jit::load() (deprecated but functional)
AOTInductor .pt2 files via AOTIModelPackageLoader (recommended)
ExecuTorch .pte files via ExecuTorch runtime (optional, edge deployment)
Auto Auto-detect from file extension

Serialization format / inference backend for model execution.

enum TensorDType

Enumerator Value Description
Float32 0 torch::kFloat32
Float64 1 torch::kFloat64
Byte 2 torch::kByte (uint8)
Int32 3 torch::kInt32
Int64 4 torch::kInt64

Tensor data types supported for slot descriptors.

Mapped to torch::ScalarType at runtime.

enum SlotDirection

Enumerator Value Description
Input
Output

Direction of a tensor slot relative to the model.

enum InterpolationMode

Enumerator Value Description
Nearest Round to nearest grid location (fast)
Bilinear Sub-pixel bilinear interpolation via grid_sample (accurate)

Interpolation strategy for spatial feature extraction.
