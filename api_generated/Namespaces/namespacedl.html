Enumerator Value Description

AOTInductor .pt2 files via AOTIModelPackageLoader (recommended)
ExecuTorch .pte files via ExecuTorch runtime (optional, edge deployment)

using BatchMode

using dl::BatchMode = typedef std::variant<FixedBatch, DynamicBatch, RecurrentOnlyBatch>;

Discriminated union describing batch-size constraints for a model.

Models report their BatchMode via [ModelBase::batchMode()](/api_generated/Classes/classdl_1_1ModelBase.md#function-batchmode). The widget uses this to configure the batch-size spinbox and select the correct weights variant.
