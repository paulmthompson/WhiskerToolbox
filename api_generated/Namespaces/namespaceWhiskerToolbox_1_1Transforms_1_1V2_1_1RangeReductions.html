function firstPositiveLatency

template <typename Element >
inline float firstPositiveLatency(
    std::span< Element const > events
)

Find time of first event with positive time.

Parameters:

  • events Range of events (should be sorted by time)

Template Parameters:

  • Element Event element type (must have time() accessor returning numeric type)

Return: Time of first positive-time event, or NaN if none found

Note: Input should be sorted by time for meaningful results. After NormalizeTime transform, t=0 is the alignment point.

Returns the time of the first event where time() > 0. This is useful for computing first-spike latency after trial alignment.

function registerAllRangeReductions

void registerAllRangeReductions()

Register all built-in range reductions.

This is called automatically at static initialization time. You typically don’t need to call this manually unless you’re working with a custom registry instance.

function registerEventRangeReductions

void registerEventRangeReductions()

Register event-based range reductions (EventWithId)

Registers:

  • EventCount: Count total events
  • FirstPositiveLatency: Time of first event after t=0
  • LastNegativeLatency: Time of last event before t=0
  • EventCountInWindow: Count events in time window
  • MeanInterEventInterval: Mean interval between events
  • EventTimeSpan: Duration from first to last event

function registerValueRangeReductions

void registerValueRangeReductions()

Register value-based range reductions (TimeValuePoint)

Registers:

  • MaxValue: Maximum value
  • MinValue: Minimum value
  • MeanValue: Mean value
  • StdValue: Standard deviation
  • TimeOfMax: Time of maximum value
  • TimeOfMin: Time of minimum value
  • TimeOfThresholdCross: First time crossing threshold
  • SumValue: Sum of values
  • ValueRange: Max - Min
  • AreaUnderCurve: Trapezoidal integration
  • CountAboveThreshold: Samples above threshold
  • FractionAboveThreshold: Fraction above threshold

function registerIntervalRangeReductions

void registerIntervalRangeReductions()

Register interval-based range reductions (IntervalWithId)

Registers:

  • IntervalCount: Count of intervals in gathered range
  • IntervalStartExtract: Start time of first interval
  • IntervalEndExtract: End time of first interval
  • IntervalSourceIndex: Entity ID of first interval
  • EventPresence: Whether any events exist (1 or 0)

function timeOfThresholdCross

template <typename Element >
inline float timeOfThresholdCross(
    std::span< Element const > points,
    ThresholdCrossParams const & params
)

Find first time when value crosses threshold.

Parameters:

  • points Range of value points (should be sorted by time)
  • params Threshold and direction parameters

Template Parameters:

  • Element Value element type (must have time() and value() accessors)

Return: Time of first crossing, or NaN if no crossing found

Detects the first crossing point in the specified direction. A crossing occurs when:

  • Rising: previous value < threshold AND current value >= threshold
  • Falling: previous value >= threshold AND current value < threshold
