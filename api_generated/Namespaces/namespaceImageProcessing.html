    ImageSize image_size,
    ColormapOptions const & options
)

Apply colormap to grayscale data for display purposes only.

Parameters:

  • grayscale_data Input grayscale image data
  • image_size Dimensions of the image
  • options Colormap options containing colormap type and parameters
  • grayscale_data Input grayscale image data
  • image_size Dimensions of the image
  • options Colormap options

Return:

  • BGRA image data if colormap is applied, empty vector if not applied
  • BGR image data if colormap is applied, empty vector if not

Note: This function does not modify the input data, only creates display output
