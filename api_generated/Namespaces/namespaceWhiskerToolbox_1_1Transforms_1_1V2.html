  • input Original input TensorData (for RowDescriptor, TimeFrame)
  • valid_indices Mapping from valid observation index → original row index
  • policy NaN policy (only Propagate and Drop are relevant here)
  • n_components Number of output columns

  • Drop: Output has only the valid rows. TimeFrameIndex and Interval rows are filtered to match.
