Types Documentation

enum AlphaCurve

Enumerator Value Description
Linear Linear falloff: alpha = max - (max-min) * (d / half_width)
Exponential Exponential falloff: faster fade near edges.
Gaussian Gaussian falloff: smooth bell-curve fade.

Alpha curve types for temporal distance → opacity mapping.

Controls how opacity falls off with temporal distance from the center of an onion-skin window.

enum GlyphType

Enumerator Value Description
Circle Filled circle marker.
Square Filled square marker.
Tick Vertical line (for event raster plots)
Cross ‘+’ shaped marker
Diamond Diamond (rotated square) — future renderer support.
Triangle Triangle marker — future renderer support.

Marker shape for discrete point/event rendering.

This is the user-facing glyph type that gets serialized to JSON. It is a superset of all widget-specific glyph enums and maps 1:1 to RenderableGlyphBatch::GlyphType values (plus additional types that may be added to the renderer in the future).

reflectcpp serializes this as a string (e.g., “Circle”, “Tick”).

enum HistogramDisplayMode

Enumerator Value Description
Bar Filled rectangles (one per bin)
Line Polyline connecting bin centers.

Rendering mode for histogram visualization.

enum HitType

Enumerator Value Description
None No hit detected.
DigitalEvent Hit a discrete event marker (has EntityId)
IntervalBody Hit inside an interval (has EntityId)
IntervalEdgeLeft Hit the left edge of an interval (for dragging)
IntervalEdgeRight Hit the right edge of an interval (for dragging)
AnalogSeries Hit within an analog series region (no EntityId)
Point Hit a point marker (has EntityId)
PolyLine Hit a polyline segment (may have EntityId)
SeriesRegion Hit within a series’ allocated region but not on data.

Types of hit targets that can be detected.

Categorizes the different kinds of elements that can be clicked/hovered in a time-series plot. The hit type determines what actions are available (e.g., interval edges can be dragged, events can be selected).

enum SeriesType

Enumerator Value Description
Analog Analog time series (continuous signals)
DigitalEvent Digital event series (discrete time points)
DigitalInterval Digital interval series (time ranges)

Type of data series for layout purposes.

AxisMapping trialIndexAxis(
    std::size_t trial_count
)

Trial index axis for raster / event plots.

Parameters:

  • trial_count Number of trials (must be > 0)

Maps world Y ∈ [-1, 1] → trial ∈ [0, trial_count). domain = (world + 1) / 2 * trial_count world = domain / trial_count * 2 - 1 Labels are integer trial indices: “0”, “1”, “2”, …
